# DHCTasks
It contains all the tasks of Developer Hub Corporation Internship

Task 1: Exploring and Visualizing a Dataset
Objective: Load, inspect, summarize, and extract visual context from the historic Iris dataset profile.  
Dataset Source Engine: Built-in seaborn Iris Flower records framework database.  
Key Engineering Findings: Explicit spatial data segmentation shows that the Setosa flower species presents significantly smaller, distinct localized petal dimension boundaries, making it linearly separable from Versicolor and Virginica species metrics across standard distribution configurations. 

Task 2: Stock Price Prediction
Objective: Implement time-series engineering regressions to predict next-day closing equity asset values.  
Dataset Source Engine: yfinance download interface pull of 2 Years of Apple (AAPL) market data.  
Model Applied: Scikit-Learn Linear Regression Algorithm.  Evaluation Performance Metrics: 
Evaluated tracking close proximity fitments metrics outputs across Mean Absolute Error ($MAE$) and Variance Coefficient ($R^2$) indicators.  

Task 3: Heart Disease Prediction
Objective: Classify patient binary health tracking states determining clinical heart condition risk statuses.  
Dataset Source Engine: Kaggle Heart Disease UCI clinical metrics repository records.  
Model Applied: Scikit-Learn Logistic Regression Pipeline initialized with Standard Feature Scaling transformations.  
Performance Metrics: Evaluated across Confusion Matrix distributions mapping true positives against false parameters and ROC-AUC curve spatial boundary calculations.  

Task 4: General Health Query Chatbot
Objective: Design and engineer an API-driven safety filtering layer system to handle medical contextual queries safely via systematic rule boundaries.  
Dataset / Model Backbone Engine: Mistral-7B-Instruct framework hosted on Hugging Face Inference server systems.  
Core Logic Implementations: Formulated highly constrained prompt directives establishing explicit rule profiles rejecting diagnostics or prescriptions, augmented with keyword safety array checks to deny self-harm/abuse inquiries.

Task 5: Mental Health Support Chatbot
Objective: Execute custom parameter fine-tuning paradigms on language learning weights to build empathetic contextual assistant responses.  
Dataset Source Engine: Facebook AI's empathetic_dialogues training dataset frame rows.  
Model Architecture Baseline: Hugging Face distilgpt2 causal transformer language weights. 
Training Compute Parameters: Configured execution under half-precision floating-point formats (fp16=True) optimizing memory footprint across Google Colab T4 environments.

Task 6: House Price Prediction
Objective: Construct robust multi-variable feature regression algorithms mapping real estate properties assets directly onto market prices.  
Dataset Source Engine: Kaggle Advanced House Prices Regression database collection records.  
Model Applied: Advanced Ensemble Gradient Boosting Regressor (GradientBoostingRegressor).  
Key Engineering Findings: Global feature importance matrix mappings proved that overall structure construction metrics classifications (OverallQual) and living area volume (GrLivArea) dictate properties pricing variances.
